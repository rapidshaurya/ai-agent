How to Use These Icons

To use these as your app icons:

  1. Take a screenshot of the icon you prefer
  2. Open it in an image editor
  3. Save it in PNG format
  4. Replace the existing icon files in the Tauri app's icons directory
  5. Rebuild the application to apply the new icons

For more detailed instructions, see the ICON_INSTRUCTIONS.md file.
